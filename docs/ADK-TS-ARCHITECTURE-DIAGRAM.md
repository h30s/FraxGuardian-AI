## ğŸ¯ ADK-TS vs Manual Implementation

### Manual (Before)
```typescript
// 1. Manual perception call
const perceptionData = await perception.perceive();

// 2. Manual reasoning call
const reasoningResult = await reasoning.reason(perceptionData);

// 3. Manual decision check
if (reasoningResult.decision === 'EXECUTE') {
    await action.execute(reasoningResult.opportunity);
}

// 4. Manual logging
console.log('Decision:', reasoningResult.decision);
```

### ADK-TS (After)
```typescript
// 1. Agent autonomously orchestrates everything
const response = await agent.run(
    "Scan for Frax Finance arbitrage and decide"
);

// 2. Agent uses tools as needed
// 3. Agent makes decision
// 4. Agent explains reasoning

// We just parse the output
const decision = parseDecision(response.content);
if (decision === 'EXECUTE') {
    await action.execute(opportunity);
}
```
